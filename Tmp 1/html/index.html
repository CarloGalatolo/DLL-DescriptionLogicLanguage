Introduzione

SDLL è un linguaggio di programmazione logica il cui scopo è la definizione e modellazione di basi di conoscenze, usando la teoria della Description Logic (logica descrittiva, DL). Un altro obiettivo, inoltre, è quello di poter creare dei file con estensione .dl, al fine di poter modellare tali basi di conoscenze come se si stesse scrivendo un programma in Python, condividendo con esso la natura di linguaggio interpretato. I consumatori finali di tale linguaggio possono usufruirne per poter avere una descrizione visiva, ed immediata, della loro base di conoscenza non appena il documento verrà eseguito dal programma, verificando al contempo correttezza sintattica e validità della base stessa. Il linguaggio è semplice, immediato, oltre che facilmente leggibile dall’utente, e con elementi sintattici comuni ad altri linguaggi (C, C++, Java).
